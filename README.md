# 禮物兌換系統

為 **陳怡安** 專屬打造的禮物兌換追蹤平台。

## 系統規格

| 項目 | 規格 |
|------|------|
| 前端框架 | Vanilla JS |
| 資料庫 | Firebase Realtime Database |
| 部署平台 | GitHub Pages |
| 目標裝置 | iPhone (iOS Safari) |

## 兌換項目

| ID | 品項 | 初始數量 | 狀態 |
|----|------|----------|------|
| `bag` | 精品包包 | 1 | Active |
| `drink` | 手搖飲料 (50嵐/迷客夏/青山) | 100 | Active |

## 功能特性

- **即時同步**: 資料透過 Firebase 即時儲存於雲端
- **跨裝置存取**: 任何裝置皆可存取最新狀態
- **單向操作**: 僅支援扣減，不可逆向增加
- **確認機制**: 每次兌換需二次確認，防止誤觸

## 技術架構

```
├── index.html          # 主程式 (HTML + CSS + JS)
├── README.md           # 專案說明文件
└── Firebase
    └── Realtime DB     # 雲端資料儲存
```

## 使用說明

1. 點擊目標品項的 `−` 按鈕
2. 於確認視窗中選擇「確定兌換」
3. 系統將自動更新雲端資料庫
